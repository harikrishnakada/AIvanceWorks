My Recommendation: Option A

A (Split Box + Mask-Fade) matches your homepage's box style, makes the image blend seamlessly via CSS mask-image (no overlays that muddy the photo), and packs content densely with inline metrics below CTAs. The key changes from current:

1. Box wrapper: rounded-2xl border border-white/[0.06] overflow-hidden — matches homepage HeroSection
2. Image blend: mask-image: linear-gradient(to right, transparent, black 30%) — image dissolves into dark bg from left edge
3. Metrics inline: Below CTAs with a subtle top border — fills the "empty space"
4. No empty space: Tighter padding, metrics fill the gap that existed below CTAs
