  • Tech chips only: accent-50 bg, accent-100 border, accent-400/accent-500 text

  • Stat cards: brand-50 bg, brand-100 border, brand-600 number
  • Platform chips: brand-50 bg, brand-100 border, brand-600 text
  • Text: text-heading, text-body, text-muted, text-subtle only
  • Borders: border-light, g100, g200 — standard gray scale
  • Zero external colors — no rose, cyan, emerald, amber, or any non-theme palette
